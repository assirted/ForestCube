# ForestCube

## Concept
Forest Cube menceritakan seekor babi lucu yang tersesat di hutan. Babi tersebut harus
melewati beberapa stage untuk sampai ke peternakan.

## Overview
Genre: Isometric Camera, survival
Platform: PC
